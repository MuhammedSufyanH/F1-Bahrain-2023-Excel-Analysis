# F1 Bahrain Grand Prix 2023 - Race Data Analysis 

## Project Overview
A comprehensive race data analysis of the 2023 Bahrain Grand Prix using Microsoft Excel.
The analysis covers 5 Drivers across 57 Laps and replicates the kind of performance engineering analysis conducted by formula 1 teams on race weekends.

## Dataset 
-285 rows of race data
-14 columns including lap times, sector times, tyre compounds, pit stop data, and gap to leader
-Drivers analysed: Verstappen, Hamilton, Alonso, Leclerc, Norris

## Analysis Performed 
1. Data cleaning and Verification
2. Lap time analysis - average pace, fastest lap, consistency (std deviation)
3. Tyre degradation analysis - SLOPE calculations, scatter chart with trendline
4. Pit stop strategy analysis - undercut effectiveness, total time lost
5. Sector prformance breakdown - theoretical best lap calculation
6. PivotTable race summary -  compound comparison, race phase analysis
7. One-page race analysis dashboard

## Key Findings 
- Verstappen set the fastest lap of 92.878s on lap 2 on fresh soft tyres
- Soft compound degraded at 0.128 s/lap for Verstappen vs 0.096 for Norris
- 6 out of 10 pit stops showed pace gained through the undercut
- Verstappen was only 0.119s away from  a theoretical perfect lap
- Hamilton's Medium-only strategy resulted in the slowest fastest lap (94.662s)

 ## Tools Used
 - Microsoft Excel 2021
 - PivotTables, AVERAGEIFS, MINIFS, SLOPE, XLOOKUP, Conditional Formatting
